# Basic-Python-for-beginners
A simple and beginner friendly Python repository. This is made for absolute beginners who are just starting Python and want clear examples, practice files, and confidence.

## Who this repo is for :
Students starting Python from zero
Beginners who feel Python is confusing
Anyone who wants clean, easy examples

## How to use this repo:
Open one folder at a time
Read the comments in code
Run the file and experiment
Change values and see what happens
